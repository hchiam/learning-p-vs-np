# Learning P vs NP

Just one of the things I'm learning. <https://github.com/hchiam/learning>

P = theoretically _solvable_ in O(n^k) time.

NP = theoretically _verifiable_ in O(n^k) time.

P ⊆ NP, but is P = NP? It seems so far that P ≠ NP.
